# TFG_OCPP_CS_QT
Sistema de control OCPP en C++, amb interficie d'usuari local amb QT.

# A.2. Execució del sistema

En cas que es vulgui obrir i executar el projecte amb QT Creator, s’han d’instal·lar les llibreries necessàries. Per fer-ho cal descraregar els fitxers `pre_install.sh` i `ocpp-cs-1.0-amd64.deb`, els quals es poden trobar en el següent repositori: [Respositori OCPP_CS](https://github.com/SergioAO8/TFG_OCPP_CS). En aquets també s'indica com s'han d'instal·lar.
